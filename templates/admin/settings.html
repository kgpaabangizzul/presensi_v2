📖 Panduan Setup

📧 Gmail App Password:

  1. Aktifkan 2-Factor Authentication di akun Google
  2. Buka myaccount.google.com/apppasswords
  3. Pilih "Mail" → "Other" → beri nama "Presensi"
  4. Salin 16-karakter password yang muncul
  5. Isi di kolom App Password di atas

💬 WhatsApp via Fonnte:

  1. Daftar di fonnte.com
  2. Tambah device → scan QR dengan WhatsApp
  3. Salin token dari dashboard
  4. Isi di kolom Token API di atas
  5. Pastikan WA yang di-scan selalu online
